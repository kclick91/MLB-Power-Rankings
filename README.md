# ⚾ MLB Power Rankings

A software system that generates dynamic power rankings for all 30 Major League Baseball teams, combining win-loss record, run differential, and divisional standing into a composite score. Further work with artificial intelligence algorithms is coming. LLMs were used to assist in this project. The Latest Rankings are displayed in this README file.
